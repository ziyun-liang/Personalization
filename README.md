# NYT Personalization Prototypes

Standalone prototype hub for personalization features. Uses QuickStart conventions.

## Setup

1. Open `personalization-features.html` directly in a browser.
2. VPN required for CDN fonts (NYT network).
3. Tokens link to `../QuickStart/tpl/tokens.css` — keep QuickStart as a sibling folder.

## Features

All features accessible via the left sidebar. Click to switch between them.
